# Pengujian Efek Carousel Embla

Ini adalah dokumen untuk menguji efek carousel Embla.

Beberapa gambar yang berurutan akan ditampilkan sebagai carousel, dan pengguna dapat beralih tampilan dengan mengklik panah atau menyeret gambar.
