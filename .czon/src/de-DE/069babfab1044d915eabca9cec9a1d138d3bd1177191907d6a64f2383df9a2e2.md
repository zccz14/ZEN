# Embla Karussell-Effekt Test

Dies ist ein Dokument zum Testen der Embla-Karussell-Effekte.

Mehrere aufeinanderfolgende Bilder werden als Karussell angezeigt. Benutzer können die Ansicht wechseln, indem sie auf Pfeile klicken oder die Bilder ziehen.

![Bild1](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)
![Bild2](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)
![Bild3](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)

Wenn jedoch Textinhalte dazwischen eingefügt werden, wird die Kontinuität des Karussells unterbrochen und es bildet sich eine neue Karussell-Gruppe.
