# Test dell'Effetto Carosello Embla

Questo è un documento per testare l'effetto carosello di Embla.

Immagini consecutive multiple verranno visualizzate come un carosello. Gli utenti possono cambiare vista cliccando sulle frecce o trascinando le immagini.

![Immagine1](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)

![Immagine3](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)

Se del contenuto testuale viene inserito nel mezzo, interromperà la continuità del carosello, formando un nuovo gruppo di caroselli.
