# Test des effets de carrousel Embla

Ceci est un document pour tester les effets de carrousel Embla.

Plusieurs images consécutives sont affichées sous forme de carrousel. Les utilisateurs peuvent changer de vue en cliquant sur les flèches ou en faisant glisser les images.

![Image1](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)
![Image2](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)
![Image3](czon://3bf2cd6040060d95c43633a99e74f42293b853944580523018311c8d95d9c441)

Si du contenu textuel est inséré au milieu, cela interrompt la continuité du carrousel, formant ainsi un nouveau groupe de carrousel.
