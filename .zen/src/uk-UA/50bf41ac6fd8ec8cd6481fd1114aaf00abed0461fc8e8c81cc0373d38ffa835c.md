# Тестування діаграм Mermaid

## Приклад блок-схеми

```mermaid
graph TD
    A[Початок] --> B{Продовжити?}
    B -->|Так| C[Виконати дію]
    B -->|Ні| D[Кінець]
    C --> E[Перевірити результат]
    E --> F{Успішно?}
    F -->|Так| G[Завершено]
    F -->|Ні| H[Повторити]
    H --> C
    G --> D
```
