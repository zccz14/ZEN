# 快速开始指南

本文档介绍如何快速开始使用 ZEN 文档生成器。

## 安装

```bash
npm install -g zengen
```

## 基本用法

1. 创建文档目录
2. 编写 Markdown 文件
3. 运行构建命令

```bash
zengen build --src ./docs --out ./dist
```

## 特性

- 极简配置
- 内置响应式模板
- AI 辅助元数据提取
- 多语言支持
