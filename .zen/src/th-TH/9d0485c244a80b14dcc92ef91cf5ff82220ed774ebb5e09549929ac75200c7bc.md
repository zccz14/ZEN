# เคล็ดลับการใช้

## การประกาศวันที่ในเนื้อหา

ในเนื้อหาสามารถใช้การประกาศวันที่เช่น "วันนี้คือ 8 มกราคม 2026" (รูปแบบอิสระ) ZEN จะจดจำและประมวลผลโดยอัตโนมัติ

เอกสาร Markdown แบบดั้งเดิมประกาศวันที่ผ่าน YAML Frontmatter ซึ่งต้องจำฟิลด์ที่เกี่ยวข้อง มิฉะนั้นอาจทำให้ไม่สามารถระบุวันที่ได้อย่างถูกต้อง

ZEN รองรับการประกาศวันที่โดยตรงในเนื้อหา ซึ่งช่วยหลีกเลี่ยงปัญหาการลืมกรอกฟิลด์ Frontmatter

AI จะดึงข้อมูลและอนุมานข้อมูลวันที่จากเนื้อหาโดยอัตโนมัติ และนำไปประมวลผลเป็นวันที่สร้างเอกสาร

วิธีนี้มีความยืดหยุ่นและชาญฉลาดมากขึ้น ช่วยยกระดับประสบการณ์ผู้ใช้

นอกจากนี้ เอกสารที่มีวันที่สามารถสร้างมุมมองไทม์ไลน์ได้ ช่วยให้การเรียกดูและการจัดการสะดวกยิ่งขึ้น เหมาะอย่างยิ่งสำหรับประเภทเนื้อหาที่ต้องการไทม์ไลน์ เช่น ไดอารี่ บล็อก
